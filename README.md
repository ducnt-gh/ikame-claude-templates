# iKame Claude Templates

Community templates for [iKame Claude Code](https://marketplace.visualstudio.com/items?itemName=ikame.ikame-claude-code) VS Code extension.

## Structure

```
agents/    → Agent definitions (~/.claude/agents/)
skills/    → Skill workflows (~/.claude/skills/)
rules/     → Coding standards (~/.claude/skills/)
```

## Contributing

Add your .md files to the appropriate folder and submit a PR.
